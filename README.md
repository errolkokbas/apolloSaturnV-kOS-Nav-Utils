# apolloSaturnV-kOS-Nav-Utils

Steering controller:

This script controls the engine gymbals on stage I & II, preventing any outboard engine from ever angling inward. In order to use this script you must follow the directions at the top of the file to properly assign the kOS id tags to your engine parts in VAB before launching craft. 


CSM RCS controller:

Automatically selects which rcs thruster nozzles to use for what axis, dependant on whether the LEM is currently docked. In order to use this script you must follow the directions at the top of the file to properly assign the kOS id tags to your engine parts in VAB before launching craft.
